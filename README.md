# Dự án Website Nội Thất SLAND

Đây là đồ án website bán hàng nội thất được xây dựng bằng PHP thuần và MySQL.

## Hướng dẫn cài đặt (Dành cho XAMPP)

1. **Tải mã nguồn:**
   - Clone repo này về thư mục `htdocs` của XAMPP.

2. **Cài đặt Database:**
   - Mở phpMyAdmin (`http://localhost/phpmyadmin`).
   - Tạo một database mới tên là: `tên_database_giống_trong_db_php`.
   - Chọn database vừa tạo, vào tab **Import (Nhập)**.
   - Chọn file `database.sql` có sẵn trong thư mục code này và bấm **Go**.

3. **Cấu hình:**
   - Mở file `db.php`, kiểm tra xem username/password đã khớp với XAMPP của bạn chưa (Mặc định là root/rỗng).

4. **Sử dụng:**
   - Truy cập: `http://localhost/ten-thu-muc-du-an/index.php`
   - Tài khoản Admin: `admin@sland.com` / `(mật khẩu của bạn)`
   - Tài khoản Khách: `(tài khoản mẫu)`

## Công nghệ sử dụng
- HTML5, CSS3, JavaScript.
- PHP Thuần.
- MySQL Database.
